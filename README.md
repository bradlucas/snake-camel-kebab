# snake-camel-kebab

snake-camel-kebab, a Clojure cmdline application

Convert between various cases. https://en.wikipedia.org/wiki/Letter_case#Special_case_styles

Snake 
Camel
Kebab


## Usage

- java -jar target/snake-camle-kebab -[sck] input-string
